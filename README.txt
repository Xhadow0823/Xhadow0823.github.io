『動くE.C.M.2』について

起動方法
ブラウザでindex.htmlを開いてください

※dataフォルダ内のSWFファイル、PNGファイルなどを直接開くこともできます
※動画は1380*960ピクセル（一部除く）より大きく表示すると画質が劣化します
※キャラクターのボイス(音声)は収録されておりません
※18歳未満の方の閲覧を禁止します
※本ソフトフェアの転載、複製を禁止します
※本製品により生じた損害には責任を負いかねます

動作環境
動作確認済みOS：WindowsXP、vista、7（64bit版動作確認済み）
※iOSでは動作しません
動作確認済みブラウザ：InternetExproler、Firefox
（PCによっては条件を満たしていても動作が重くなる場合があります）
